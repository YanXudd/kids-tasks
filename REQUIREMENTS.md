# 功能需求文档

## 1. 用户系统

### 1.1 注册
- 角色选择：家长/孩子
- 用户名 + 密码（bcrypt 加密）
- 孩子注册需填写家庭邀请码
- 头像：选择 emoji 表情

### 1.2 登录
- 用户名 + 密码登录
- JWT Token 认证
- Token 有效期管理

### 1.3 权限
- 家长：管理任务、审核打卡、管理商品、查看统计
- 孩子：打卡、查看积分、兑换商品

## 2. 家庭系统

### 2.1 家庭创建
- 家长注册时自动创建家庭
- 生成唯一邀请码（6位大写字母）

### 2.2 成员管理
- 孩子通过邀请码加入家庭
- 一个家庭可有多个孩子
- 家长可查看所有孩子信息

## 3. 任务系统

### 3.1 任务类型
- 奖励任务：完成后获得正积分
- 减分任务：触发后扣除积分

### 3.2 任务属性
- 名称
- Emoji 图标
- 分类（可选）
- 积分值（正/负）
- 重复规则

### 3.3 重复规则
- 每日：每天重复
- 每周：选择周一到周日
- 每月：选择几号
- 固定日期：指定具体日期

### 3.4 任务分配
- 可分配给特定孩子
- 不指定则分配给所有孩子

## 4. 打卡系统

### 4.1 孩子打卡
- 查看今日任务列表
- 选择任务提交打卡
- 支持批量提交

### 4.2 家长审核
- 查看待审核打卡列表
- 确认：积分自动累加
- 拒绝：记录拒绝原因

### 4.3 打卡状态
- 待提交：未选择
- 已提交：等待审核
- 已确认：积分已发放
- 已拒绝：打卡无效

## 5. 积分系统

### 5.1 积分获取
- 完成奖励任务
- 家长手动加分

### 5.2 积分扣除
- 触发减分任务
- 家长手动扣分

### 5.3 积分消费
- 兑换商品

### 5.4 积分流水
- 记录所有积分变动
- 包含时间、类型、金额、描述

## 6. 商品系统

### 6.1 商品属性
- 名称
- 图片
- 价格（积分）
- 库存（可设为无限）
- 分类（可选）
- 单位和数量（可选）

### 6.2 库存管理
- 有限库存：售罄后自动下架
- 无限库存：始终可用
- 单品库存：每次兑换扣减

### 6.3 分类管理
- 自定义分类
- 按分类筛选商品

## 7. 兑换系统

### 7.1 孩子兑换
- 浏览商品列表
- 选择商品兑换
- 积分自动扣除

### 7.2 订单管理
- 待兑现：等待家长购买
- 已兑现：商品已交付
- 已撤回：积分退还

### 7.3 家长操作
- 查看兑换订单
- 标记已兑现
- 撤回订单（退还积分）

## 8. 统计系统

### 8.1 积分统计
- 当前积分
- 累计获得
- 累计扣除
- 累计消费

### 8.2 打卡统计
- 今日完成数
- 趋势图表（周/月/年）

### 8.3 孩子排名
- 按积分排序
- 查看各孩子详情

## 9. 设置系统

### 9.1 家庭设置
- 查看/复制邀请码
- 修改密码

### 9.2 系统设置
- 切换家长/孩子视图
- 退出登录

## 10. 界面设计

### 10.1 设计风格
- 卡通可爱风格
- 橙色为主色调
- 圆角卡片设计
- 动画过渡效果

### 10.2 响应式
- 手机端优先
- 平板适配
- 桌面端居中显示

### 10.3 交互设计
- 左右滑动切换分类
- 上下滚动查看更多
- 点击反馈动画
- 确认对话框

## 11. 数据备份

### 11.1 备份内容
- 数据库
- 代码文件
- 配置文件
- 上传图片

### 11.2 备份策略
- 每日备份
- 保留最近 3 份
- 自动清理旧备份

## 12. 安全要求

### 12.1 认证安全
- 密码 bcrypt 加密
- JWT Token 认证
- Token 过期机制

### 12.2 数据安全
- SQL 注入防护
- XSS 防护
- 文件上传限制

### 12.3 权限控制
- 角色权限分离
- 接口权限验证
- 数据访问控制
