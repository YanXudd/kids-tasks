# 📸 截图说明

在此目录添加应用截图，README.md 中会引用这些图片。

## 建议截图

1. **login.png** — 登录/注册页面
2. **tasks.png** — 任务列表页面（儿童端）
3. **checkin.png** — 打卡确认页面
4. **points.png** — 积分详情页面
5. **shop.png** — 奖励商店页面
6. **admin-tasks.png** — 任务管理页面（家长端）
7. **admin-review.png** — 打卡审核页面
8. **admin-products.png** — 商品管理页面
9. **admin-stats.png** — 数据统计页面

## 截图方法

1. 在浏览器中打开应用
2. 按 F12 打开开发者工具
3. 切换到手机视图（iPhone/iPad）
4. 截图各页面
5. 保存到此目录
